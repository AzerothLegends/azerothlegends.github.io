Guia básico ELUNA

Essa explicação cobre os conceitos básicos que você pode explorar durante a sua aprendizagem e compreensão de como manipular o comportamento do servidor usando Eluna Lua Engine no AzerothCore.

Pré-requisitos

Para começar você precisa das seguintes coisas:

  • Um editor de texto (Notepad++, VSCode..)
  • Um servidor Azerothcore compilado com ELUNA LUA ENGINE

Instruções

  1. Variáveis em Eluna Lua Engine

    2. Variáveis em Lua são usadas para armazenar dados que podem ser manipulados ao longo do código. Elas podem ser de vários tipos, como números, strings, tabelas, etc. Em Eluna, você pode usar variáveis para armazenar informações sobre personagens, criaturas, ou até mesmo o estado do jogo.

    local playerName = "AzerothHero"
    -- Uma string armazenada na variável playerName

    local playerLevel = 70
    -- Um número armazenado na variável playerLevel
  2. Funções no Eluna Lua Engine.

    Funções são blocos de código que podem ser chamados para executar uma tarefa específica. Elas ajudam a organizar o código e a reutilizar trechos de código sem repetições.

    function GreetPlayer(player)
       player:SendBroadcastMessage("Bem-vindo ao Azeroth!")
    end
  3. Chamando uma função:
    4. GreetPlayer(player)

    Neste exemplo, a função 'GreetPlayer' recebe um objeto 'player' e envia uma mensagem de boas-vindas para ele.

  4. Register Event no Eluna Lua Engine

    5. Register Event é uma funcionalidade que permite vincular uma função Lua a um evento específico no servidor, como quando um jogador entra no jogo, ataca uma criatura, ou usa um item. Isso permite que o desenvolvedor adicione comportamentos personalizados ao jogo sem precisar modificar o código-fonte principal.

    local function OnPlayerLogin(event, player)
       player:SendBroadcastMessage("Você se conectou com sucesso!")
    end
    RegisterPlayerEvent(3, OnPlayerLogin)

    Neste exemplo, a função 'OnPlayerLogin' é registrada para o evento de login do jogador (código de evento 3). Sempre que um jogador fizer login, a mensagem será enviada.

  5. Resumo

    6. Variáveis: Armazenam dados que podem ser manipulados.

    Funções: Executam tarefas específicas e podem ser reutilizadas.

    Register Event: Vincula uma função a um evento do servidor, permitindo adicionar comportamento personalizado ao jogo.
